Remainder and Factor Theorem: Full Tutorial

Learn how to solve polynomial division using the Remainder and Factor Theorem!

Step 1: Understanding the Remainder Theorem

The Remainder Theorem states that when a polynomial f(x) is divided by (x - c), the remainder is f(c). This means that the remainder of the division is just the result of substituting c into the polynomial.

For example, if you divide f(x) = x³ - 6x² + 11x - 6 by (x - 2), we can find the remainder by simply calculating f(2).

Example:

Let's find the remainder when f(x) = x³ - 6x² + 11x - 6 is divided by (x - 2).

f(2) = 2³ - 6 * 2² + 11 * 2 - 6 = 8 - 24 + 22 - 6 = 0

The remainder is 0, so (x - 2) is a factor of f(x).

Step 2: Understanding the Factor Theorem

The Factor Theorem states that if f(c) = 0(x - c) is a factor of the polynomial f(x).

So, in the example above, because f(2) = 0, (x - 2) is a factor of x³ - 6x² + 11x - 6.

Example:

If f(x) = x³ - 6x² + 11x - 6 and f(2) = 0, then (x - 2) is a factor of f(x).

f(x) = (x - 2)(x² - 4x + 3)

This is how we factor the polynomial!
